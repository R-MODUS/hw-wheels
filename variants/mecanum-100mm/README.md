## Parametry

| Parametr | Hodnota |
| --- | --- |
| Průměr kola | 100 mm |
| Šířka kola | 46 mm |
| Šířka s průchodkou motoru | 56 mm |
| Počet válečků | 9 |
| Otvor hřídele | Ø 6 mm |
| Hmotnost | ~0,33 kg |
| Materiál ráfku / dílů | PLA (3D tisk) |
| Povrch válečků | polyolefin |
| Ložiska | 623 |

## Obsah sestavy

- vnitřní a vnější prstenec (L i R / mirrored)
- válečky + ložiska 623
- svěrka hřídele (`prt_3_4_shaft_clamp`)
- spojovací materiál (M3/M4, heat inserty)

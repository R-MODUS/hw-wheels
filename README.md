# hw-wheels

Wheels in this repository belong to the [R-MODUS](https://github.com/R-MODUS) project, but they can be reused in other robots as well.

All variants are currently designed for a **Ø 6 mm** motor shaft.

This repo is **mechanical only** — no firmware or ROS.

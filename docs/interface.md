# Rozhraní — kola (společné)

Platí pro všechny varianty v tomto repu, pokud varianta neurčí jinak.

## Mechanika

| Parametr | Požadavek / poznámka |
| --- | --- |
| Hřídel motoru | typicky Ø 6 mm (D-cut dle motoru) |
| Upevnění | svěrka hřídele (`shaft clamp`) + spojovací materiál |
| Orientace mecanum | zrcadlové páry L/R — viz dokumentace varianty |
| Montáž na rám | přes motor clamp platformy / vlastní držák |

## Elektrika / SW

Žádné — čistě mechanický díl.

## Kompatibilita s R-MODUS

Referenční build používá `variants/mecanum-100mm` na [hw-robot-platform](https://github.com/R-MODUS/hw-robot-platform) (varianta `mecanum-reference`).
